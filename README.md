# Grid Naming Convention

##### A1: Aircraft Cell
##### A0: Aircraft Shotted Cell
##### B1: Battleship Cell
##### B0: Battleship Shotted Cell
##### C1: Cruiser Cell
##### C0: Cruiser Shotted Cell
##### S1: Submariner Cell
##### S0: Submariner Shotted Cell
##### D1: Destroyer Cell
##### D0: Destroyer Shotted Cell
##### O0: Empty Cell
##### X0: Shotted Empty Cell
##### K1: Clickable Cell
##### K0: Clicked Cell
